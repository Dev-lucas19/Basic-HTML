Conceitos básicos de HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Ela define a estrutura do conteúdo na web.

Elementos HTML

Os elementos HTML são os blocos de construção de uma página web. Cada elemento é definido por uma tag, que pode ter atributos.

Exemplo de Elemento

<h1>Título Principal</h1>

O exemplo acima cria um título principal na página.

Estrutura Básica de um Documento HTML

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
</head>
<body>
    <h1>Título Principal</h1>
    <p>Este é um parágrafo de exemplo.</p>
</body>
</html>

Tags Comuns

  • <h1> a <h6>: Títulos de diferentes níveis.
  • <p>: Define um parágrafo.
  • <a href="URL">: Cria um link para outra página.
  • <img src="URL" alt="Descrição">: Insere uma imagem.

Conclusão

HTML é essencial para a criação de páginas web. Compreender sua estrutura básica é o primeiro passo para se tornar um desenvolvedor web.
